# LHBot v0.0.1
Basic Discord Bot made by Lewis Hughes.<br>
## prerequisites
discord.py<br>
#### Linux/macOS
'python3 -m pip install -U discord.py'
#### Windows
'py -3 -m pip install -U discord.py`<br>
discord developer account<br>
>enter api token on bottom line of 'main.py'<br>
## commands
>please note that default prefix for commamnds is '.' but can be changed in main.py.<br>
#### say
removes the user's message containing what is to be said, then displays it below.<br>
#### purge
removes the content and message history of channel executed in<br>
> please note that I have not yet added user permission checks so this will work for all users until checks are added<br>
